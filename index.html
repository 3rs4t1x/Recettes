Sync GitHub Gist

Sauvegarde cloud gratuite, multi-appareil

1
Créer un token GitHub

Va sur github.com/settings/tokens/new, donne un nom (ex : "Carnet"), coche uniquement gist, clique Generate token et copie-le.

2
Colle ton token ici
3
C'est tout !

Le carnet se synchronise automatiquement. Sur un autre appareil, répète ces étapes avec le même token.
